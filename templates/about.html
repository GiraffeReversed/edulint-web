About the page

This page was built for computer science students at the Faculty of Informatics, Masaryk University to serve as a simple linter to help them overcome the most common programmer-beginner's mistakes.

The project is available to everybody, but no guarantees of uptime are provided.

Behind the page

The backend is powered by Flask. The frontend is designed using Bootstrap, with bootstrap-dark-5 for the dark mode. The icons are the Bootstrap Icons. The editor is provided by CodeMirror, the themes by CodeMirror Themes. The background image is taken from Transparent textures. Split.js is used for window splitting. The favicon is from the Noun Project.

The analysis uses flake8 and pylint.

This page was built by Anna Řechtáčková. Project's GitHub .
